# Security Policy

## Reporting a Vulnerability

If you believe you have found a security issue, please report it privately.

Preferred method:
1. Open a GitHub Security Advisory for this repository.
2. Include clear reproduction steps, impact, and any suggested fix.

If a private report is not possible, open a GitHub issue and avoid posting
exploit details.

## Supported Versions

We recommend using the latest release available. Security fixes are applied to
the most recent version.
